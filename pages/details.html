How I made this website?

There are various way you can create and manage a website. Basically, a website is created by HTML, CSS, Javascript, SQL and many more. While you create the structure of your webpage using HTML (Hypertext Markup Language), and then add styling by using CSS (Cascading Style Sheets). You may add some other behaviour in your webpage by using scripting language such as Javascript. This way, you can create a website. For this you may need a coding skill of basic programming.

On the other hand you can use some online tools to create a website, for instance Elementor. Also, Wordpress (A CMS: Content Management System) will help you to create a website.

Frequently Asked Question (FAQ)

  1. Q: What is Domain?
    Ans: domains are like server addresses where your website or web applications are available. A domain name can be any combination of letters and numbers. Each domain is unique and has an extension for example .com, .xyz, and many more.

  2. Q: What is Hosting?
    Ans: Web hosting is an online service that allows your website's content accessible on the internet from any where in the world. When you purchase a hosting plan, it means you are renting space on a physical server to store all the website's files and data.

  3. Q: What is Netlify?
    Ans: Netlify is a remote-first cloud computing company that offers a development platform that includes build, deploy, and serverless backend services for web applications and dynamic websites.

  4. Q: What is Piprahost
    Ans: A domain and hosting provider site, based on Bangladesh.
